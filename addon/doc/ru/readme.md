# <p align="center">xPlorer</p>

<br>

<p align="center">Улучшите работу с Проводником Windows с помощью продвинутой автоматизации и удобных инструментов навигации.</p>

## <p align="center">Описание</p>

**xPlorer** — это комплексный пакет инструментов для пользователей NVDA, специально разработанный, чтобы сделать Проводник Windows умнее и эффективнее. Он устраняет сложности в повседневном управлении файлами, автоматизируя повторяющиеся задачи. Будь то пакетное создание папок, стандартизация имен или извлечение текста из файлов без их открытия, xPlorer обеспечивает бесперебойную работу профессионального уровня для каждого пользователя. Это возможность делать больше с меньшим количеством нажатий клавиш.

<br>

## <p align="center">Что нового</p>

• **Создание нескольких папок:** Мгновенно создавайте целую структуру каталогов из одного списка.  
<br>
• **Преобразование регистра для папок:** Стандартизируйте имена папок (ВЕРХНИЙ регистр, нижний, Начинать С Прописных, Как в предложениях).  
<br>
• **Подробная информация о папке:** Получайте размер и количество элементов в реальном времени прямо из меню.  
<br>
• **Оптимизация заголовков:** Скрытие лишнего «— Проводник» в заголовках окон для более чистой речи.  
<br>
• **Мгновенное создание папки:** Автоматическое использование содержимого буфера обмена в качестве имени при создании новой папки.

<br>

## <p align="center">Горячие клавиши</p>

> **NVDA+Shift+X** : Открыть контекстное меню xPlorer  
> (Центральный хаб для всех профессиональных функций, включая смену регистра и пакетное создание папок)

> **NVDA+Shift+Z** > • **Одинарное нажатие** : Сообщить размер (Объявить общий размер выбранных элементов)  
> • **Двойное нажатие** : Сжать в Zip (Архивация выбранных файлов с интеллектуальным именованием)

> **NVDA+Shift+C** > • **Одинарное нажатие** : Копировать имена (Копировать имена выбранных файлов или папок в буфер обмена)  
> • **Двойное нажатие** : Копировать путь текущей папки (Получить полный путь к текущей папке)

> **NVDA+Shift+V** > • **Одинарное нажатие** : Копировать содержимое (Извлечь текстовое содержимое прямо из выбранного файла)  
> • **Двойное нажатие** : Инвертировать выделение (Быстрое переключение фокуса между выбранными и невыбранными элементами)

> **NVDA+Shift+F2** : Переименовать только файл  
> (Фокусируется только на имени, защищая расширение файла от случайных изменений)

> **Control+Shift+N** : Создать папку с авто-вставкой  
> (Создает новую папку и мгновенно вставляет содержимое буфера обмена в качестве имени)

<br>

## <p align="center">Функции</p>

### <p align="center">1. Продвинутое пакетное создание папок</p>

Функция **«Создать несколько папок»** (в меню xPlorer) создана для серьезной организации. Вместо создания папок по одной, вы можете вставить список имен в одно диалоговое окно. xPlorer обработает весь список и мгновенно создаст каждую папку в текущем каталоге.

### <p align="center">2. Профессиональный конвертер регистра</p>

Добейтесь единообразия в вашей файловой системе. Выберите папку(и) и используйте меню xPlorer для мгновенного изменения имен:  
<br>
• **ВЕРХНИЙ РЕГИСТР:** Переводит всё в заглавные буквы (например, «ДАННЫЕ ПРОЕКТА»).  
<br>
• **нижний регистр:** Переводит всё в строчные буквы (например, «данные проекта»).  
<br>
• **Регистр Заголовков:** Каждое слово с большой буквы (например, «Данные Проекта»).  
<br>
• **Как в предложениях:** Только самая первая буква заглавная (например, «Данные проекта»).

### <p align="center">3. Умное создание «Из буфера в папку»</p>

С xPlorer создание папок становится одношаговым процессом. При нажатии **Control+Shift+N** дополнение проверяет буфер обмена и автоматически подставляет имя. Больше никакого ручного переименования — просто скопируйте, нажмите клавишу, и готово.

### <p align="center">4. Извлечение содержимого файла</p>

Экономьте время, извлекая текст без открытия приложений. **Одинарное нажатие NVDA+Shift+V** копирует текст из файла прямо в буфер обмена. Это отлично работает с различными текстовыми форматами.

### <p align="center">5. Интеллектуальное Zip-архивирование</p>

По **двойному нажатию NVDA+Shift+Z** xPlorer упаковывает выделенное в zip-файл. Он предотвращает потерю данных, проверяя существующие файлы и добавляя числовой суффикс. Звуковые сигналы сообщают о прогрессе.

### <p align="center">6. Панель настроек xPlorer</p>

Настройте работу через **Настройки NVDA > xPlorer**:  
<br>
<br>
• **Автоматически выбирать первый элемент:** При входе в папку фокус сразу ставится на первый файл.  
<br>
• **Объявлять «Пустая папка»:** Четкое голосовое подтверждение, что в папке ничего нет.  
<br>
• **Скрывать класс DirectUIHWND:** Убирает технический мусор из речи NVDA.  
<br>
• **Скрывать «— Проводник» в заголовках:** Помогает быстрее идентифицировать папки на слух.
