# <p align="center">xPlorer</p>

<br>

<p align="center">ยกระดับการจัดการไฟล์บน Windows File Explorer ด้วยระบบอัตโนมัติและเครื่องมืออัจฉริยะที่เหนือกว่าต้นฉบับ</p>

<br>

<p align="center"><b>ผู้เขียน:</b> chai chaimee</p>
<p align="center"><b>URL:</b> https://github.com/chaichaimee/xPlorer</p>

## <p align="center">คำอธิบาย</p>
**xPlorer** คือปรากฏการณ์ใหม่ในการจัดการไฟล์สำหรับผู้ใช้ NVDA ที่ต้องการความเร็วและความแม่นยำสูงสุด เราเปลี่ยน Windows File Explorer ให้กลายเป็นเวิร์กสเตชันที่ทรงพลังโดยการกำจัดขั้นตอนที่ซ้ำซ้อนและน่าเบื่อออกไป ไม่ว่าคุณจะเป็นโปรแกรมเมอร์ที่จัดการโปรเจกต์ขนาดใหญ่หรือผู้ใช้ทั่วไปที่ต้องการความเป็นระเบียบ xPlorer จะช่วยให้คุณจัดการข้อมูลมหาศาลได้ในเสี้ยววินาที ด้วยปุ่มลัดที่ออกแบบตามหลักสรีรศาสตร์จริง

<br>

## <p align="center">มีอะไรใหม่</p>
• **ระบบ Robo:** ปฏิวัติการคัดลอกและย้ายไฟล์ด้วยเอนจินพิเศษที่ทำงานเร็วกว่าระบบมาตรฐานของ Windows กว่า 50%  
<br>
• **การประมวลผลไฟล์โดยตรง:** ระบบประมวลผลไฟล์แบบ Direct ที่ช่วยลดการใช้ทรัพยากรเครื่องและเพิ่มความเสถียรเมื่อต้องจัดการกับข้อมูลจำนวนมาก

<br>

## <p align="center">ปุ่มลัด (Hot Keys)</p>

> **NVDA+Shift+R (Robo Copy & Move)**  
> • **กดครั้งเดียว:** Robo Copy (ล็อคตำแหน่งไฟล์หรือโฟลเดอร์เป้าหมาย)  
> • **กดสองครั้ง:** Robo Paste (วางไฟล์/โฟลเดอร์ที่คัดลอกไปยังปลายทางทันที)  
> • **กดสามครั้ง:** **Robo Move (คำสั่งย้ายไฟล์ด่วน - ดึงข้อมูลจากต้นทางมาที่นี่ด้วยความเร็วสูงสุด)**

> **NVDA+Shift+X** : เมนู xPlorer (ศูนย์รวมเครื่องมือจัดการชื่อขั้นสูงและเครื่องมือสร้างโฟลเดอร์)

> **NVDA+Shift+Z**  
> • **กดครั้งเดียว:** บอกขนาด (คำนวณและแจ้งขนาดรวมของไฟล์ที่เลือกทั้งหมดอย่างแม่นยำ)  
> • **กดสองครั้ง:** บีบอัด Zip (รวมไฟล์ที่เลือกเป็น .zip พร้อมตั้งชื่อให้อัตโนมัติ)

> **NVDA+Shift+C**  
> • **กดครั้งเดียว:** คัดลอกชื่อที่เลือก (คัดลอกเฉพาะชื่อไฟล์เพื่อนำไปใช้งาน)  
> • **กดสองครั้ง:** คัดลอกแถบที่อยู่ (คัดลอกพาธของโฟลเดอร์ปัจจุบัน)

> **NVDA+Shift+V**  
> • **กดครั้งเดียว:** คัดลอกเนื้อหา (ดึงข้อความภายในไฟล์เข้าสู่คลิปบอร์ดโดยไม่ต้องเปิดไฟล์)  
> • **กดสองครั้ง:** สลับการเลือก (เลือกไฟล์ที่เหลือในโฟลเดอร์สลับกับที่เลือกอยู่)

> **NVDA+Shift+F2** : เปลี่ยนชื่อเฉพาะไฟล์ (เปลี่ยนชื่อไฟล์โดยยังคงนามสกุลไฟล์ไว้อย่างปลอดภัย)

> **Control+Shift+N** : สร้างโฟลเดอร์พร้อมตั้งชื่ออัตโนมัติ (สร้างโฟลเดอร์ใหม่และตั้งชื่อตามข้อความในคลิปบอร์ดทันที)

<br>

## <p align="center">คุณสมบัติเด่น</p>

### 1. ระบบ Robo: พลังการย้ายไฟล์ระดับอุตสาหกรรม
บอกลาความล่าช้าของ Ctrl+C และ Ctrl+V แบบเดิมๆ ด้วย **ระบบ Robo** ที่ออกแบบมาเพื่อการจัดการไฟล์ปริมาณมากโดยเฉพาะ
* **ความเร็วที่เหนือกว่า:** เมื่อย้ายไฟล์หรือโฟลเดอร์ขนาดใหญ่ หรือโฟลเดอร์ที่มีไฟล์ย่อยนับพัน ระบบ Robo จะใช้ตรรกะการเข้าถึงพาธไฟล์โดยตรง ลดเวลาการประมวลผลลงกว่าครึ่งเมื่อเทียบกับคำสั่งมาตรฐานของ Windows
* **Robo Move (กดสามครั้ง):** นี่คือไม้ตายสำหรับการจัดระเบียบข้อมูล เพียงกดสามครั้ง ระบบจะ 'ดึง' ข้อมูลจากต้นทางมาวางที่ปลายทางทันทีด้วยความเร็วสูงสุด เหมาะสำหรับโปรเจกต์ที่ต้องย้ายข้อมูลข้ามไดรฟ์หรือข้ามโฟลเดอร์บ่อยๆ

### 2. การสร้างโฟลเดอร์ชุดขั้นสูง
ฟีเจอร์นี้ออกแบบมาเพื่อลดงานที่ทำซ้ำๆ หากคุณต้องการสร้างโฟลเดอร์ 10, 20 หรือ 100 โฟลเดอร์พร้อมกัน เพียงเปิดเมนู **"สร้างโฟลเดอร์จำนวนมาก"** แล้ววางรายการชื่อ ระบบจะเขียนโครงสร้างโฟลเดอร์ลงดิสก์พร้อมกันในครั้งเดียว ช่วยให้คุณเซ็ตอัพโปรเจกต์ใหม่ได้ในเวลาไม่ถึง 2 วินาที

### 3. ตัวแปลงรูปแบบตัวอักษรระดับมืออาชีพ
ความเป็นมืออาชีพวัดกันที่ความใส่ใจในรายละเอียด xPlorer ช่วยให้คุณจัดระเบียบชื่อไฟล์และโฟลเดอร์ให้เป็นรูปแบบเดียวกันทั้งระบบ ไม่ว่าจะเป็น **ตัวพิมพ์ใหญ่ทั้งหมด** สำหรับเอกสารสำคัญ หรือ **ตัวพิมพ์ใหญ่หน้าคำ** เพื่อความสวยงาม คุณสามารถเปลี่ยนชื่อไฟล์นับร้อยได้ด้วยการกดเมนูเพียงครั้งเดียว

### 4. การดึงเนื้อหาไฟล์ในทันที
ในวันที่คุณต้องการเช็คข้อมูลด่วน xPlorer ให้คุณเข้าถึงเนื้อหาภายในไฟล์ที่เป็นข้อความได้ทันทีโดยไม่ต้องรอเปิด Notepad หรือโปรแกรมอ่านไฟล์อื่นๆ เพียงกด **NVDA+Shift+V** เนื้อหาภายในจะถูกดึงออกมาให้คุณอ่านหรือนำไปวางต่อได้ทันที ลดภาระของ CPU และประหยัดเวลาของคุณ

### 5. เวิร์กโฟลว์อัจฉริยะ "จากคลิปบอร์ดสู่โฟลเดอร์"
ผสานการทำงานระหว่างการรับข้อมูลและการจัดเก็บเข้าด้วยกัน เมื่อคุณคัดลอกชื่อโปรเจกต์จากอีเมลหรือแชท เพียงกด **Control+Shift+N** ใน File Explorer ระบบจะสร้างโฟลเดอร์ใหม่และตั้งชื่อนั้นให้ทันที ลดขั้นตอนการกด F2 และวางชื่อให้เหลือเพียงการกระทำเดียว

### 6. การเก็บถาวร Zip ที่ชาญฉลาด
การบีบอัดไฟล์จะไม่เป็นเรื่องยุ่งยากอีกต่อไปด้วยระบบ **Smart Zip** ที่ตรวจสอบการซ้ำกันของชื่อไฟล์โดยอัตโนมัติ หากมีชื่อซ้ำ ระบบจะรันตัวเลขต่อท้ายเพื่อป้องกันการเขียนทับข้อมูลสำคัญ พร้อมระบบแจ้งเตือนด้วยเสียง (Beep codes) ที่ทำให้คุณทราบสถานะการทำงานโดยไม่ต้องมองหน้าจอ
