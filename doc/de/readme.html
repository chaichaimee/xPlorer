Neue Tastenkombinationen bearbeiten – für bequemere Bedienung und Verbesserung der Funktion „Größe vorlesen“


Übersicht

xPlorer ist ein Werkzeug, das den Zugriff auf Operationen im Datei-Explorer deutlich erleichtert. Die neueste Version enthält Funktionen zum Verwalten von Dateien und Ordnern, wie das Komprimieren von Elementen in eine ZIP-Datei, das Umbenennen von Dateien über ein eigenes Fenster, das Dateiname und Erweiterung trennt, um versehentliches Löschen zu verhindern, das Vorlesen der Größe von Dateien, Ordnern oder Laufwerken sowie das Kopieren der Namen ausgewählter Dateien und Ordner in die Zwischenablage usw.


Hotkeys


• NVDA+Shift+X : xPlorer-Kontextmenü öffnen

Wählen Sie eine Datei oder einen Ordner aus und öffnen Sie das Kontextmenü. Dieses Untermenü fasst alle xPlorer-Funktionen an einem Ort zusammen, sodass Sie jede Funktion sofort und bequem nutzen können.

• NVDA+Shift+Z Einfacher Druck : Größe vorlesen

Wählen Sie eine Textdatei im Editor (die bereits Inhalt hat) oder mehrere Dateien (jede nicht größer als 10 MB). Drücken Sie die Tastenkombination, um den gesamten Inhalt zu kopieren, ohne die Dateien zu öffnen – spart viele Schritte bei mehreren Dokumenten. In der neuesten Version können Sie auch mehrere Dateien oder Ordner auswählen, um die Gesamtgröße zu erfahren.


• NVDA+Shift+Z Doppeldruck : Ausgewählte Elemente in ZIP komprimieren

Wählen Sie die Dateien oder „Alles auswählen“, dann drücken Sie die Tastenkombination. Die ausgewählten Elemente werden automatisch in eine ZIP-Datei komprimiert.


• NVDA+Shift+C : Namen ausgewählter Dateien und Ordner in die Zwischenablage kopieren

Wählen Sie die gewünschten Dateien (eine oder mehrere) oder „Alles auswählen“, dann drücken Sie die Tastenkombination. Alle Namen der ausgewählten Dateien und Ordner werden in die Zwischenablage kopiert, um sie in einem Texteditor einzufügen.


• NVDA+Shift+C Doppeltipp : Adressleiste im Datei-Explorer kopieren

Wenn Sie sich im gewünschten Ordner befinden, drücken Sie einfach doppelt die Tastenkombination, um die Adressleiste direkt zu kopieren – schneller als Strg+L zu verwenden.

• NVDA+Shift+V Einfacher Druck : Inhalt kopieren

Wählen Sie eine Textdatei (die bereits Inhalt hat) oder mehrere Dateien (jede nicht größer als 10 MB), dann drücken Sie die Tastenkombination. Sie können den gesamten Inhalt kopieren, ohne die Dateien zu öffnen. Unterstützte Dateiendungen: .txt, .rtf, .py, .js, .html, .css, .xml, .json, .csv, .md, .ini, .conf, .cfg, .java, .cpp, .c, .h, .php, .rb, .pl, .sh, .bat, .ps1


• NVDA+Shift+V Doppeltipp : Auswahl umkehren

Wählen Sie eine Datei oder einen Ordner aus, drücken Sie NVDA+Shift+I, um die Auswahl umzukehren (nicht ausgewählte Elemente werden markiert). Ein erneutes Drücken kehrt die Auswahl wieder um.

• NVDA+Shift+F2 : Ausgewählte Datei umbenennen

Wählen Sie eine Datei aus und drücken Sie die Tastenkombination. Es öffnet sich ein Fenster mit getrennten Feldern für Dateiname und Dateierweiterung. Sie können Name oder Erweiterung einfach bearbeiten. Drücken Sie Enter, um das Fenster zu schließen. Nur der geänderte Teil wird automatisch aktualisiert.


Funktionen

Zusätzliche Funktionen aus früheren Versionen können ein- oder ausgeschaltet werden über: NVDA-Menü → Einstellungen → Einstellungen → xPlorer


  • Automatisch das erste Element auswählen
  • „Leerer Ordner“ ansagen, wenn ein leerer Ordner geöffnet wird
  • Ankündigung der Klasse DirectUIHWND unterdrücken (aktiviert)


Datei erstellen

Während der Fokus im Datei-Explorer am gewünschten Ort liegt, öffnen Sie das xPlorer-Kontextmenü und wählen „Datei erstellen“. Es erscheint ein Dialog, in dem Sie den gewünschten Dateinamen und die Erweiterung festlegen können. Sie können beliebig viele Dateien erstellen lassen. Das Feature unterstützt viele Text- und Code-Dateiformate.

Robocopy ist ein Befehl zum „Verschieben“ oder „Kopieren“ von Ordnern und Dateien – intelligenter als die normale Windows-Kopierfunktion.

Gleiche Dateien werden nicht erneut kopiert, wenn sie identisch sind.

Selbst bei sehr vielen oder sehr großen Dateien kann Robocopy bei Unterbrechung genau dort fortsetzen, wo es aufgehört hat.

Kurz gesagt: Ideal, wenn Sie große Datenmengen „verschieben“ möchten, ohne jedes Mal von vorne zu beginnen.


Spiegel-Backup mit Zeitplan

Diese Funktion ermöglicht automatische Spiegel-Backups, die zu einer von Ihnen gewählten Zeit ablaufen – alle paar Minuten, Stunden, Tage oder Wochen.

Zur festgelegten Zeit führt Robocopy automatisch ein Backup des Quellordners in den Zielordner durch und erstellt eine exakte Spiegelkopie der neuesten Daten.

Eine einfache und zuverlässige Methode, Ihre Dateien stets aktuell zu halten – ohne manuelles Eingreifen.


TXT zu Ordner

Wählen Sie eine .txt-Datei, die eine Liste enthält, drücken Sie NVDA+Alt+X und wählen Sie die Funktion „TXT zu Ordner“. xPlorer erstellt dann einen Unterordner mit dem Namen der .txt-Datei und legt darin für jeden Listeneintrag einen eigenen Ordner an.
