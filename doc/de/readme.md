## Übersicht

xPlorer ist ein Werkzeug, das den einfachen Zugriff auf Datei-Explorer-Operationen ermöglicht.  
Neueste Version enthält Funktionen wie ZIP-Komprimierung, sicheres Umbenennen (Name und Erweiterung getrennt), Größenanzeige, Kopieren von Namen in die Zwischenablage usw.

## Tastenkombinationen

- **NVDA+Shift+X** : xPlorer-Kontextmenü öffnen  
- **NVDA+Shift+Z** (einmal tippen) : Größe vorlesen  
- **NVDA+Shift+Z** (doppelt tippen) : Ausgewählte Elemente zu ZIP komprimieren  
- **NVDA+Shift+C** : Namen der ausgewählten Dateien/Ordner kopieren  
- **NVDA+Shift+C** (doppelt tippen) : Adressleiste kopieren  
- **NVDA+Shift+V** (einmal tippen) : Inhalt kopieren (unterstützt viele Text-/Code-Formate)  
- **NVDA+Shift+V** (doppelt tippen) : Auswahl umkehren  
- **NVDA+Shift+F2** : Ausgewählte Datei umbenennen (getrennte Felder für Name/Erweiterung)

(ส่วนต่อไปแปลคล้ายต้นฉบับ ใช้โครงสร้างเดียวกัน – ถ้าต้องการส่วนไหนเพิ่มเติม บอกได้ครับ)
