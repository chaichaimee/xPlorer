## Обзор

xPlorer упрощает работу с Проводником Windows: архивация в ZIP, безопасное переименование, чтение размеров, копирование имён и т.д.

## Горячие клавиши

- **NVDA+Shift+X** : Открыть контекстное меню xPlorer  
- **NVDA+Shift+Z** (одно нажатие) : Сказать размер  
- **NVDA+Shift+Z** (двойное нажатие) : Сжать выбранное в ZIP
