Обзор

xPlorer — это инструмент, созданный для более удобного доступа к операциям в Проводнике. Последняя версия включает функции управления файлами и папками: сжатие элементов в ZIP, переименование файлов через отдельное окно с разделением имени и расширения (чтобы случайно не удалить расширение), чтение размера файлов, папок или дисков, копирование имён выбранных файлов и папок в буфер обмена и многое другое.


Горячие клавиши


• NVDA+Shift+X : Открыть контекстное меню xPlorer

Выделите файл или папку и откройте контекстное меню. Это подменю объединяет все функции xPlorer в одном месте для быстрого доступа.

• NVDA+Shift+Z Одиночное нажатие : Сказать размер

Выделите текстовый файл в редакторе (с уже имеющимся содержимым) или несколько файлов (каждый не больше 10 МБ). Нажмите сочетание, чтобы скопировать всё содержимое без открытия файлов — экономит много действий при работе с несколькими документами. В новой версии также можно выделить несколько файлов или папок, чтобы узнать общий размер.


• NVDA+Shift+Z Двойное нажатие : Сжать выбранные элементы в ZIP

Выделите файлы или выберите всё, затем нажмите сочетание. Выбранные элементы автоматически сжимаются в ZIP-архив.


• NVDA+Shift+C : Скопировать имена выбранных файлов и папок в буфер обмена

Выделите нужные файлы (один или несколько) или всё, затем нажмите сочетание. Все имена выбранных файлов и папок копируются в буфер обмена для вставки в текстовый редактор.


• NVDA+Shift+C Двойное нажатие : Скопировать адресную строку Проводника

Находясь в нужной папке, просто дважды нажмите сочетание, чтобы сразу скопировать адресную строку — быстрее, чем открывать её через Ctrl+L.

• NVDA+Shift+V Одиночное нажатие : Скопировать содержимое

Выделите текстовый файл (с содержимым) или несколько файлов (каждый ≤ 10 МБ), затем нажмите сочетание. Можно скопировать всё содержимое без открытия файлов. Поддерживаемые расширения: .txt, .rtf, .py, .js, .html, .css, .xml, .json, .csv, .md, .ini, .conf, .cfg, .java, .cpp, .c, .h, .php, .rb, .pl, .sh, .bat, .ps1


• NVDA+Shift+V Двойное нажатие : Инвертировать выделение

Выделите файл или папку, нажмите NVDA+Shift+I, чтобы инвертировать выделение (выделяются невыделенные элементы). Повторное нажатие возвращает исходное состояние.

• NVDA+Shift+F2 : Переименовать выбранный файл

Выделите файл и нажмите сочетание. Откроется окно с отдельными полями для имени и расширения. Можно легко изменить имя или расширение. Нажмите Enter для закрытия. Автоматически обновится только изменённая часть.

Создание файла

Находясь в Проводнике в нужном месте, откройте контекстное меню xPlorer и выберите «Создать файл». Появится диалог для указания имени и расширения. Можно создать любое количество файлов. Поддерживается множество текстовых и программных форматов.

Robocopy — команда для «перемещения» или «копирования» папок и файлов, более умная, чем обычное копирование Windows.

Не копирует повторно одинаковые файлы.

Даже при большом количестве или очень больших файлах может продолжить с места прерывания.

Идеально подходит для перемещения больших объёмов данных без необходимости начинать заново.


Зеркальное резервное копирование по расписанию

Эта функция позволяет настроить автоматическое **зеркальное резервное копирование**, которое запускается в выбранное вами время — каждые несколько минут, часов, дней или недель.

В назначенное время Robocopy автоматически создаёт зеркальную копию папки-источника в папке-приёмнике с самыми свежими данными.

Простой и надёжный способ поддерживать файлы в актуальном состоянии без ручной работы.


TXT в Папку

Выделите .txt-файл со списком, нажмите NVDA+Alt+X и выберите функцию TXT в Папку. xPlorer создаст подпапку с именем .txt-файла, а внутри — отдельную папку для каждого пункта списка.
