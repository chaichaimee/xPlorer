Aperçu

xPlorer est un outil conçu pour faciliter l’accès aux opérations de l’Explorateur de fichiers. La dernière version inclut des fonctionnalités de gestion de fichiers et dossiers, comme compresser des éléments en ZIP, renommer des fichiers via une fenêtre dédiée séparant nom et extension pour éviter les suppressions accidentelles, lire la taille de fichiers, dossiers ou disques, copier les noms des fichiers et dossiers sélectionnés dans le presse-papiers, etc.


Raccourcis clavier


• NVDA+Shift+X : Ouvrir le menu contextuel xPlorer

Sélectionnez un fichier ou un dossier et ouvrez le menu contextuel. Ce sous-menu regroupe toutes les fonctions xPlorer au même endroit pour un accès rapide.

• NVDA+Shift+Z Appui simple : Dire la taille

Sélectionnez un fichier texte dans l’éditeur (déjà avec contenu) ou plusieurs fichiers (aucun > 10 Mo). Appuyez sur le raccourci pour copier tout leur contenu sans ouvrir les fichiers – évite de nombreuses étapes avec plusieurs documents. Dans la dernière version, vous pouvez aussi sélectionner plusieurs fichiers ou dossiers pour connaître la taille totale.


• NVDA+Shift+Z Appui double : Compresser les éléments sélectionnés en ZIP

Sélectionnez les fichiers ou tout sélectionner, puis appuyez sur le raccourci. Les éléments sélectionnés seront automatiquement compressés dans un fichier ZIP.


• NVDA+Shift+C : Copier les noms des fichiers et dossiers sélectionnés dans le presse-papiers

Sélectionnez les fichiers souhaités (un ou plusieurs) ou tout sélectionner, puis appuyez sur le raccourci. Tous les noms seront copiés dans le presse-papiers pour les coller dans un éditeur de texte.


• NVDA+Shift+C Double appui : Copier la barre d’adresse de l’Explorateur

Quand vous êtes dans le dossier dont vous voulez copier le chemin, appuyez deux fois sur le raccourci pour copier directement la barre d’adresse – plus rapide que Ctrl+L.

• NVDA+Shift+V Appui simple : Copier le contenu

Sélectionnez un fichier texte (avec contenu) ou plusieurs fichiers (≤ 10 Mo chacun), puis appuyez sur le raccourci. Vous pouvez copier tout le contenu sans ouvrir les fichiers. Extensions supportées : .txt, .rtf, .py, .js, .html, .css, .xml, .json, .csv, .md, .ini, .conf, .cfg, .java, .cpp, .c, .h, .php, .rb, .pl, .sh, .bat, .ps1


• NVDA+Shift+V Appui double : Inverser la sélection

Sélectionnez un fichier ou dossier, appuyez sur NVDA+Shift+I pour inverser la sélection (surligner les éléments non sélectionnés). Un nouvel appui revient à la sélection précédente.

• NVDA+Shift+F2 : Renommer le fichier sélectionné

Sélectionnez un fichier et appuyez sur le raccourci. Une fenêtre s’ouvre avec des champs séparés pour le nom et l’extension. Vous pouvez modifier facilement le nom ou l’extension. Appuyez sur Entrée pour fermer. Seule la partie modifiée est mise à jour automatiquement.


Fonctionnalités

Les fonctionnalités des versions précédentes peuvent être activées ou désactivées via : Menu NVDA → Préférences → Paramètres → xPlorer

Créer un fichier

Lorsque le focus est dans l’Explorateur à l’emplacement souhaité, ouvrez le menu contextuel xPlorer et choisissez Créer un fichier. Une boîte de dialogue permet d’indiquer le nom et l’extension. Vous pouvez créer autant de fichiers que vous voulez. Compatible avec de nombreux formats texte et programmation.

Robocopy est une commande pour « déplacer » ou « copier » dossiers et fichiers de façon plus intelligente que la copie Windows classique.

Elle ne recopie pas les fichiers identiques.

Même avec beaucoup ou de très gros fichiers, elle reprend là où elle s’est arrêtée en cas d’interruption.

Parfait pour déplacer de grandes quantités de données sans devoir tout recommencer.


Sauvegarde miroir programmée

Cette fonction permet de configurer des **sauvegardes miroir automatiques** qui s’exécutent à l’heure choisie — toutes les quelques minutes, heures, jours ou semaines.

À l’heure prévue, Robocopy sauvegarde automatiquement le dossier source vers le dossier destination en créant une copie miroir des données les plus récentes.

Une méthode simple et fiable pour garder vos fichiers à jour sans intervention manuelle.


TXT vers Dossier

Sélectionnez un fichier .txt contenant une liste, appuyez sur NVDA+Alt+X et choisissez la fonction TXT vers Dossier. xPlorer crée un sous-dossier portant le nom du fichier .txt et y place un dossier par élément de la liste.
