## Quoi de neuf dans la dernière version ?

- Créer un fichier
- Modification des raccourcis clavier : plus pratique et amélioration de la fonction Dire la taille

## Aperçu

xPlorer facilite les opérations dans l'Explorateur de fichiers : compression ZIP, renommage sécurisé, lecture de tailles, copie de noms, etc.

## Raccourcis clavier

- **NVDA+Shift+X** : Ouvrir le menu contextuel xPlorer  
- **NVDA+Shift+Z** (simple appui) : Dire la taille
