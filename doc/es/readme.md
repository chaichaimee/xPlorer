## Resumen

xPlorer facilita las operaciones en el Explorador de Archivos. Incluye compresión ZIP, renombrado seguro (separando nombre y extensión), lectura de tamaños, copiar nombres al portapapeles, etc.

## Atajos de teclado

- **NVDA+Shift+X** : Abrir menú contextual xPlorer  
- **NVDA+Shift+Z** (toque simple) : Decir tamaño  
- **NVDA+Shift+Z** (doble toque) : Comprimir elementos seleccionados a ZIP
