Resumen

xPlorer es una herramienta diseñada para facilitar el acceso a las operaciones del Explorador de archivos. La versión más reciente incluye funciones para gestionar archivos y carpetas, como comprimir elementos en ZIP, renombrar archivos mediante una ventana dedicada que separa nombre y extensión para evitar borrados accidentales, leer el tamaño de archivos, carpetas o unidades, y copiar los nombres de archivos y carpetas seleccionados al portapapeles, entre otras.


Atajos de teclado


• NVDA+Shift+X : Abrir menú contextual de xPlorer

Selecciona un archivo o carpeta y abre el menú contextual. Este submenú reúne todas las funciones de xPlorer en un solo lugar para acceder a ellas rápidamente.

• NVDA+Shift+Z Toca una vez : Decir tamaño

Selecciona un archivo de texto en el editor (que ya tenga contenido) o varios archivos (ninguno mayor de 10 MB). Pulsa el atajo para copiar todo su contenido sin abrir los archivos – ahorra pasos al trabajar con varios documentos. En la última versión también puedes seleccionar varios archivos o carpetas para ver el tamaño total.


• NVDA+Shift+Z Toca dos veces : Comprimir elementos seleccionados en ZIP

Selecciona los archivos o selecciona todo, luego pulsa el atajo. Los elementos seleccionados se comprimirán automáticamente en un archivo ZIP.


• NVDA+Shift+C : Copiar nombres de archivos y carpetas seleccionados al portapapeles

Selecciona los archivos deseados (uno o varios) o selecciona todo, luego pulsa el atajo. Todos los nombres de archivos y carpetas seleccionados se copiarán al portapapeles para pegarlos en un editor de texto.


• NVDA+Shift+C Doble toque : Copiar barra de direcciones del Explorador

Cuando estés en la carpeta cuyo camino deseas copiar, simplemente pulsa dos veces el atajo para copiar directamente la barra de direcciones – más rápido que usar Ctrl+L.

• NVDA+Shift+V Toca una vez : Copiar contenido

Selecciona un archivo de texto (que ya tenga contenido) o varios archivos (ninguno mayor de 10 MB), luego pulsa el atajo. Puedes copiar todo el contenido sin abrir los archivos. Extensiones soportadas: .txt, .rtf, .py, .js, .html, .css, .xml, .json, .csv, .md, .ini, .conf, .cfg, .java, .cpp, .c, .h, .php, .rb, .pl, .sh, .bat, .ps1


• NVDA+Shift+V Doble toque : Invertir selección

Selecciona un archivo o carpeta, pulsa NVDA+Shift+I para invertir la selección (resalta los elementos no seleccionados). Volver a pulsar invierte nuevamente.

• NVDA+Shift+F2 : Renombrar archivo seleccionado

Selecciona un archivo y pulsa el atajo. Se abrirá una ventana con campos separados para nombre y extensión. Puedes editar fácilmente el nombre o la extensión. Pulsa Enter para cerrar. Solo se actualizará automáticamente la parte modificada.


Funciones

Funciones adicionales de versiones anteriores se pueden activar o desactivar en: Menú NVDA → Preferencias → Configuración → xPlorer


  • Seleccionar automáticamente el primer elemento
  • Anunciar «Carpeta vacía» al entrar en una carpeta vacía
  • Suprimir anuncio de la clase DirectUIHWND (activado)


Crear archivo

Mientras el foco está en el Explorador de archivos en la ubicación deseada, abre el menú contextual de xPlorer y selecciona Crear archivo. Aparecerá un diálogo para indicar el nombre y extensión deseados. Puedes crear cualquier cantidad de archivos. Soporta muchos formatos de texto y programación.

Robocopy es un comando para «mover» o «copiar» carpetas y archivos de forma más inteligente que la copia normal de Windows.

No vuelve a copiar archivos idénticos.

Incluso con muchos archivos o archivos muy grandes, puede continuar desde donde se interrumpió.

Ideal cuando deseas mover grandes cantidades de datos sin tener que empezar de cero cada vez.


Copia de seguridad en espejo con programación

Esta función permite configurar copias de seguridad en espejo automáticas que se ejecutan en el horario que elijas — cada pocos minutos, horas, días o semanas.

En el momento programado, Robocopy respaldará automáticamente la carpeta origen en la carpeta destino, creando una copia exacta de los datos más recientes.

Una forma sencilla y confiable de mantener tus archivos actualizados sin hacerlo manualmente.


TXT a Carpeta

Selecciona un archivo .txt que contenga una lista, pulsa NVDA+Alt+X y elige la función TXT a Carpeta. xPlorer creará una subcarpeta con el nombre del archivo .txt y dentro creará una carpeta por cada elemento de la lista.
